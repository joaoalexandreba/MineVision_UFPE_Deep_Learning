# MineVision

Projeto da disciplina de Engenharia de Software da Pós-Graduação em Deep Learning.

## Objetivo

Desenvolver um sistema de visão computacional capaz de:

- Detectar caminhões de mineração;
- Contabilizar veículos em operação;
- Classificar caminhões como carregados ou vazios;
- Gerar estatísticas simples da operação.

## Dataset

https://universe.roboflow.com/minitruck/mining-truck-v1.4
